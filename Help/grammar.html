# 一些说明

# 关于 Markdown 语言

  1. Markdown 不是编程语言（因为其不涉及逻辑处理），而是描述性语言，其完成的工作本质上就是图文排版
  2. 我们 wiki 的模板已经可以做到将 Markdown 语言直接转换为网页了！因此大家只需要用 Markdown 就可以了，不用再去学 HTML+CSS+js+... 了
  3. 石墨文档也支持部分 Markdown 语法（小声）

# 关于 Typora 编辑器

  1. Typora 是一款能够即时显示效果的编辑器，适用于 Markdown 基本语法的所有内容；但我们的 wiki 使用的 Shoka 主题新增了一些特殊的语法，Typora 就不能显示了
  2. 软件下载地址：官网链接

# 其他

Shoka 主题提供的字体修饰有很多，例如波浪线、着重点、荧光高亮、彩虹等等，但由于目前美工组还在设计配色方案，因此不展开细说具体有哪些修饰方法。所有需要强调的内容请先统一使用荧光高亮标出，待配色方案确定后会协助大家修改。

  1. 所有的语法规则，都会按照 基本格式 - 实例代码 - 实例效果 的形式呈现
  2. 推荐一边看一边在 Typora 编辑器上动手操作
  3. 此处仅列出了 Markdown 中较常用的一些功能，更多功能待通过鲁棒性验证后将在本网页更新（请关注 “本页内容” 处的更新日期与版本号），建议收藏以方便查看
  4. 如有任何意见、建议或遇到困难，请找 GhoST👻

# 无序列表

格式

- 列表内容

实例

- Team
- Project
- Parts
- Safety
- Human Practice
- Awards
- Poster

效果

  • Team
  • Project
  • Parts
  • Safety
  • Human Practice
  • Awards
  • Poster

- 号与内容之间需要有一个空格

  1. 除了可以使用 - 号，还可以使用 + 号、 * 号，效果相同。但符号与内容之间都需要有一个空格
  2. 在 Typora 编辑器中，一旦有了第一个列表内容，按 Enter 就会自动生成下一个列表项目，再按 Enter 则会停止自动生成列表项目

# 表格

格式

|列1|列2|列3|...|
|对齐说明|对齐说明|对齐说明|...|
|内容|
  1. | 号即 C / Python 语言中表示 “或运算” 的符号
  2. Typora 编辑器功能比较完善，你只需输入第一行（即带有分隔符的表头）就会自动生成表格，然后可以鼠标操作
  3. 对齐说明参考如下：
    • 文字默认左对齐，即输入 -
    • 文字居中：输入:-:
    • 文字右对齐：输入 -:
    • 输入的 - 号数量不作限制，有一个就行，为了对齐可以多加几个
    • 冒号需要在 英文状态 下输入

实例

|列1|列2|列3|
|--|:--:|-:|
|1-1|1-2|1-3|
|2-1|2-2|2-3|

效果

列 1列 2列 3
1-11-21-3
2-12-22-3
